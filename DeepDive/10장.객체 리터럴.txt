10.1 객체란?

    원시 값을 제외하면 모두 객체!!
    => 원시 값은 변경 불가 그러나 객체는 변경 가능

    객체는 0개 이상의 프로퍼티로 이루어져 있다
    => 프로퍼티는 키와 값으로 구성

    ex) const person = {
        name: 'Kim',
        age: '20',
    };

    프로퍼티 값이 함수라면 메서드라고 부른다!

10.2 객체 리터럴에 의한 객체 생성

    위 예시처럼 중괄호 {}를 사용하여 객체를 생성한다
    => 객체 리터럴, 즉 값으로 평가되는 표현식이므로 ;이 붙는다!!

10.3 프로퍼티

    쉼표 ,로 구분한다
    프로퍼티 키: 문자열 또는 심벌 값
    프로퍼티 값: 자바스크립트에서 사용할 수 있는 모든 값

10.4 메서드

    자바스크립트의 함수는 일급 객체다
    => 다만 구분을 위해 프로퍼티 값이 함수라면 메서드라 칭한다

10.5 프로퍼티 접근

    - 마침표 .을 사용 
    - 대괄호 []를 사용 => 프로퍼티 키가 반드시 ''로 감싼 문자열이어야 한다!

10.6 프로퍼티 값 갱신

    변수처럼 재할당이 가능하다

10.7 프로퍼티 동적 생성

    동적으로 추가된다

10.8 프로퍼티 삭제

    delete 연산자 사용

10.9 ES6에서 추가된 객체 리터럴의 확장 기능

    (1) 프로퍼티 축약 표현
        프로퍼티 값으로 변수를 사용하는 경우, 변수 이름과 프로퍼티 키가 같다면 생략 가능

    (2) 계산된 프로퍼티 이름
        (타입 변환된) 문자열 표현식을 사용하여 프로퍼티 키를 동적 생성 가능

    (3) 메서드 축약 표현
        프로퍼티 값으로 함수가 사용될 때, function 키워드 생략 가능
