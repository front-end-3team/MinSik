4.1 변수란 무엇인가? 왜 필요한가?

    변수는 값을 저장하기 위해 확보한 메모리 공간 자체 또는 그 메모리 공간을 식별하기 위해 붙인 이름
    => 값을 참조하기 위해 값의 위치를 가리키는 이름
    변수에 값을 저장하는 것을 할당(대입, 저장)이라 하고 변수에 저장된 값을 읽으면 참조라고 한다

4.2 식별자

    식별자 === 변수 이름(메모리 주소)
    식별자에는 값이 저장되어 있는 메모리 주소를 저장(기억)

4.3 변수 선언

    변수를 생성하는 행동을 말하며 변수 사용을 위해 반드시 거쳐야 한다
    let, const 키워드를 사용
    변수를 선언만 한다면 자바스크립트 엔진에 의해 undefined 값이 암묵적으로 할당

4.4 변수 선언의 실행 시점과 변수 호이스팅

    변수 선언문이 런타임이 아니라 그 이전 단계에서 실행되어 코드 맨 위로 올려진 것처럼 동작하는 특징
    모든 선언문은 런타임 이전 단계에서 먼저 실행된다

4.5 값의 할당

    할당 연산자(=)을 사용한다
    변수 선언은 호이스팅에 의해 먼저 실행되지만 값의 할당은 런타임에 실행된다

4.6 값의 재할당

    이미 값이 할당되어 있는 변수에 새로운 값을 또다시 할당하는 것
    해당 메모리 주소에 재할당할 값이 덮어 씌어지는 것이 아닌 새로운 메모리 주소에 할당하여 가리키는 것이며 후에 가비지컬렉터에서 해제

4.7 식별자 네이밍 규칙

    특수문자를 제외한 문자, 숫자, _, $를 포함할 수 있다
    단, 문자, _, $로 시작해야 하며 숫자로 시작하는 것은 금지된다
    예약어는 식별자로 사용 불가능

    * 네이밍 컨벤션

        -카멜 케이스 ex) firstName
        -스네이크 케이스 ex) first_name
        -파스칼 케이스 ex) FirstName
