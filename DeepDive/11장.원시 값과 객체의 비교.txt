11.1 원시 값

    (1) 변경 불가능한 값
        원시 값은 변경 불가
        => 메모리 자체를 말하므로 값의 재할당이지 변경은 아니다!!

    (2) 문자열과 불변성
        유사 배열 객체이지만 원시 값 중 문자열 타입일 뿐 객체가 아니다

    (3) 값에 의한 전달
        변경 불가 => 재할당만 가능 => 사실상 다른 메모리의 주소를 가리킨다

11.2 객체

    (1) 변경 가능한 값
        프로퍼티로 구성 => 동적 추가 가능(변경 가능)
        단, 여러 개의 식별자가 하나의 객체를 공유할 수 있어서 데이터의 보안 ↓

        얕은 복사: 참조 값(주소)를 복사
        깊은 복사: 원시 값처럼 객체의 모든 값을 통째로 복사 => 완전한 복사본

    (2) 참조에 의한 전달
        사실상 자바스크립트에는 값에 의한 전달만이 존재
