31.1 정규 표현식이란?

    일정한 패턴을 가진 문자열의 집합을 표현하기 위해 사용하는 형식 언어다
    => 자바스크립트의 고유 문법이 아니며, 펄의 정규 표현식 문법이 도입되었다
    => 특정 패턴과 일치하는 문자열을 검색하거나 추출 또는 치환할 수 있는 기능인 패턴 매칭 기능을 제공한다
    => 반복문이나 조건문 없이도 간단하게 코드를 작성할 수 있으나 가독성이 좋지 않다

31.2 정규 표현식의 생성
    
    패턴과 플래그를 사용한 정규 표현식 리터털로 생성할 수 있다
    => RegExp 생성자 함수로도 RegExp 객체를 생성할 수 있다

31.3 RegExp 메서드

    (1) RegExp.prototype.exec
        패턴 매칭 결과를 배열로 전달한다

    (2) RegExp.prototype.test
        패턴 매칭 결과를 불리언 값으로 전달한다

    (3) String.prototype.match
        패턴 매칭 결과를 배열로 전달한다

31.4 플래그

    i (Ignore case): 대소문자를 구별하지 않고 패턴을 검색한다
    g (Global): 대상 문자열 내에서 패턴과 일치하는 모든 문자열을 전역 검색한다
    m (Multi line): 문자열의 행이 바뀌더라도 패턴 검색을 계속한다

31.5 패턴

    (1) 문자열 검색
        RegExp 메서드를 사용하여 대상 문자열과 매칭 결과를 구하면 검색이 실행된다
    
    (2) 임의의 문자열 검색
        .은 임의의 문자 한 개를 의미한다

    (3) 반복 검색
        {m, n} 최소 m번, 최대 n번 반복되는 문자열의 패턴을 검색한다

    (4) OR 검색
        | 사용

    (5) NOT 검색
        ^ 사용

    (6) 시작 위치로 검색
        [] 밖에서 ^ 사용

    (7) 마지막 위치로 검색
        $ 사용

31.6 자주 사용하는 정규표현식

    (1) 특정 단어로 시작하는지 검사
    
    (2) 특정 단어로 끝나는지 검사

    (3) 숫자로만 이루어진 문자열인지 검사

    (4) 하나 이상의 공백으로 시작하는지 검사

    (5) 아이디로 사용 가능한지 검사

    (6) 메일 주소 형식에 맞는지 검사

    (7) 핸트폰 번호 형식에 맞는지 검사

    (8) 특수 문자 포함 여부 검사
