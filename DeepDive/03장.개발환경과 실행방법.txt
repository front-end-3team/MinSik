3.1 자바스크립트 실행 환경

    자바스크립트는 브라우저 환경 또는 Node.js 환경에서 실행할 수 있다
    다만 용도에 맞는 환경에서 개발해야 한다

3.2 웹 브라우저

    (1) 개발자 도구
        F12 또는 Ctrl + Shift + I

    (2) 콘솔
        디버깅을 수행할 때 사용해라!!

    (3) 브라우저에서 자바스크립트 실행
        VScode에서 Alt + L/O
    
    (4) 디버깅
        에러 원인을 파악하여 해결하는 과정

3.3 Node.js

    (1) Node.js와 npm 소개
        Node.js는 브라우저 이외의 환경에서 동작시킬 수 있다
        npm은 자바스크립트 패키지 매니저다

    (2) Node.js 설치
        http://nodejs.org

    (3) Node.js REPL
        터미널을 통해 코드를 실행 가능

3.4 비주얼 스튜디오 코드

    (1) 비주얼 스튜디오 코드 설치
        https://code.visualstudio.com

    (2) 내장 터미널
        Node.js 명령어로 자바스크립트 파일을 실행 가능

    (3) Code Runner 확장 플러그인
        Ctrl + Alt + N으로 실행
        단, Node.js 환경을 사용하므로 Web API가 포함된 소스코드는 에러를 발생시킨다
        
    (4) Live Server 확장 플러그인
        소스코드를 수정할 때마다 수정 사항을 브라우저에 자동으로 반영해준다
