5.1 값

    표현식이 평가되어 생성된 결과
    => 변수에 들어가는 모든(원시타입 및 객체타입) 것이 다 값!!

5.2 리터럴

    사람이 이해할 수 있는 문자 또는 약속된 기호를 사용해 값을 생성하는 표기법
    => 사람도 척 보면 이해할 수 있고 자바스크립트 엔진 또한 런타임에서 값을 생성하게 하는 약속된 표기법

5.3 표현식

    값으로 평가될 수 있는 문
    => 표현식이 평가되면 새로운 값을 생성하거나 기존값을 참조 
    => 리터럴 또한 그 자체로 표현식이다

5.4 문

    프로그램을 구성하는 기본 단위이자 최소 실행 단위 
    => 문의 집합으로 이뤄진 것이 바로 프로그램

    토큰이란?
    문법적으로 더 이상 나눌 수 없는 코드의 기본 요소

    토큰 => 문 => 프로그램

5.5 세미콜론과 세미콜론 자동 삽입 기능

    ;은 문의 종료를 의미

5.6 표현식인 문과 표현식이 아닌 문

    변수에 할당이 가능하다면 표현식인 문이다
